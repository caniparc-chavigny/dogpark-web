# Contribuer / Contributing

Merci de votre intérêt ! / Thank you for your interest!

## Comment contribuer
1. Ouvrez une *issue* pour discuter du changement proposé.
2. Forkez le dépôt et créez une branche descriptive.
3. Soumettez une *pull request* claire, en expliquant le « pourquoi ».

## Principes du projet (non négociables)
- **Vie privée d'abord** : aucune donnée personnelle collectée ni conservée.
- **Simplicité** : le code reste lisible et minimal.
- **Bienveillance** : un outil au service de la communauté.

## Signaler un bogue
Ouvrez une *issue* avec les étapes pour reproduire. Pour une faille de
sécurité, voir [SECURITY.md](SECURITY.md).
